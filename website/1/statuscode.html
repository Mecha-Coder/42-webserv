HTTP Status Code Notes

Error Codes

  • 400: Malform request
  • 400: Invalid Header
  • 403: No default / autoIndex
  • 403: Invalid POST – Not CGI
  • 404: Resource not found
  • 405: Invalid Method
  • 410: Delete Invalid Resource
  • 413: Client body exceed limit
  • 500: No permission – read / delete
  • 500: CGI Hang
  • 500: CGI Crash

Success Codes

  • 200: Get resource
  • 201: Create resource
  • 204: Delete resource
  • 301: Normal redirect
  • 308: Redirect to add “/” at end of URI
