How to Test Virtual Hosting

Option 1: Edit /etc/hosts

For Linux: sudo nano /etc/hosts

For Windows: C:\Windows\System32\drivers\etc\hosts

Add this line and save the file:

127.0.0.1 example.com

Option 2: Use curl --resolve (CLI only, no browser)

curl --resolve example.com:8080:127.0.0.1 http://example.com/
We test using Option 2. Here are the test cases: 
curl --resolve example.com:4000:127.0.0.1 http://example.com:4000/
curl --resolve example.com:4000:127.0.0.1 http://test.com:/
